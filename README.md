# Call Center Performance Dashboard

## Objective

The objective of this dashboard is to analyze call center performance and provide a clear view of key operational metrics. The dashboard highlights how efficiently calls are being handled, how frequently issues are resolved, and how customers rate their experience. By combining several visualizations and interactive filters, the dashboard allows users to quickly explore performance trends and evaluate agent-level results.

## Dashboard Components
The dashboard is organized into several views that highlight different aspects of call center performance.

### Answered Rate
This donut chart shows the percentage of calls that were answered versus unanswered. It provides a quick overview of how effectively incoming calls are being handled.

### Resolved Rate
This chart displays the proportion of calls that were successfully resolved. It helps indicate how well the support team is addressing customer issues.

### Average Answer Speed
This KPI shows the average time it takes for an agent to answer a call. Faster response times generally indicate better service efficiency.

### Average Satisfaction Gauge
This gauge displays the overall customer satisfaction rating on a 0–5 scale. The gauge format makes it easy to quickly see how current satisfaction compares to the maximum possible rating.

### Calls Per Month
This bar chart shows the number of calls received each month, with a breakdown of answered and unanswered calls. It helps reveal call volume trends over time.

### Agent Statistics
This table summarizes individual agent performance. It includes the number of calls answered, number resolved, average satisfaction rating, and average answer speed for each agent.

### Filters
The dashboard includes filters for **Agent**, **Topic**, and **Date**. These allow users to interactively explore the data and focus on specific segments of the call center activity.

## Summary
This Tableau dashboard brings together several metrics to monitor call center activity, including answered rate, resolved rate, average answer speed, and customer satisfaction. It also includes a monthly view of call volume and a detailed agent performance table. Interactive filters allow users to examine results by agent, topic, or date range. Overall, the dashboard provides a simple way to track service quality and identify areas where operational improvements may be needed.

## Key Insights

The dashboard highlights several trends in call center performance.

- The overall **customer satisfaction rating averages 3.40 out of 5**, suggesting there may be opportunities to improve service quality or customer experience.
- Approximately **81% of incoming calls were answered**, while about **19% were missed**, indicating relatively strong call coverage but still leaving a noticeable portion of calls unattended.
- Around **73% of calls were successfully resolved**, meaning most customer issues were handled during the interaction, though there is still room to improve resolution rates.
- The **average speed of answer is about 67 seconds**, which may contribute to the lower satisfaction score if customers experience longer wait times.
- Monthly call volume remains fairly consistent, with **over 1,200 calls answered each month**, indicating a steady workload for the support team.
